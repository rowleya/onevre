Venue Monitoring

The Venue Monitoring part give you all the information to navigate within the Venue.

This Panel consists of three parts, the heading states the current venue and current activities when the venue changes.

Venue tree

On the left there is a tree of venues that can be accessed from the current Venue. This tree can be show in three modes to be selected from the listbox:

All Venues
All the venues that are available on the current Venue Server will be listed alphabetically
Exits
Venues that are subvenues of the current Venue
My Venues
A selection of venues stored by the user as "favourites"

Clicking on a venue in the tree will change into this venue

Venue Information

On the right there is tree containing the information on the current venue. This includes information on:

Participants
A list of all the participants that joined the meeting. Each listed Participant contains its profile expandable in the tree. Clicking on your entry opens the dialogue to configure your User Profile
Data
A list of data items connected to the Venue.
Services
List of Services available in the Venue
Application Sessions
Shared applications accessible from the venue. Each application listed includes the shared application monitor in this tree.
