Access Grid Configuration

The Access Grid configuration panel allows you to configure all aspects of your local node setup.

The Configuration panel consits of a Button bar that shows the current setup and permits changes to the configuration and a Navigation Bar to input the url of the new venue.

/ Audio enabled / disabled

Shows the status and enables or disables the audioService.

/ Display enabled / disabled

Shows the status and enables or disables the videoConsumerService.

/ Video enabled / disabled

Shows the status and enables or disables the videoProducerService.

Configure Node Services

Opens a Dialogue to configure your node services

Configure User Profile

Lets you configure your User Profile

Multicast / Unicast configuration

Lets you configure your bridging needs

/ Configure Encryption

Lets you configure an encrypted Session

Configure My Venues

Lets you configure your personal venues


Upload Node Service

Lets you upload a new nodeService to the PAG server

Upload Shared Application

Lets you upload a new shared application package to the PAG server

Upload Bidge

Lets you upload a new bridge to the PAG server
